Password Generator

I have been getting lots of emails lately about the heartbleed vulnerability, and I feel like it's time to pay more attention to my passwords. I've been using the same set of two or three basic passwords for a few years now. I know this is a bad thing, but I haven't got around to doing anything about it yet. To that end, I made a simple program to generate random passwords of a specified length. Click here to check it out.

Cheat Sheets!

The first set of programming cheat sheets are up. These are intended to act as quick reference sheets on programming syntax for various languages.

When I was learning programming, I often found myself wishing for a single resource to remind me of syntax defferences between languages. These reference sheets are my solution. They contain basic syntax for Javascript, C#, Java, Ruby, Perl, and Python. More may be added in the future as needed. Click here for the cheat sheets.
